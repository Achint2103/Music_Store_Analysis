# SQL_Project_Music_Store_Analysis

SQL project to analyze online music store data

This project aims to analyze the music playlist database, enabling the store to gain insights into its business growth by addressing fundamental questions.

## Database and Tools

- Postgre SQL
- PgAdmin4
